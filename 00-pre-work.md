# Battleships Workshop — Pre-Work

**Please complete these steps before arriving. Setup takes ~20 minutes.**
If you get stuck, bring your laptop and we'll sort it out on the day.

---

## 1. Create a free Honeycomb account

Honeycomb is the observability tool you'll use to see inside your bot during the workshop.

1. Go to **[honeycomb.io](https://www.honeycomb.io)** and click **Start for Free**
2. Sign up with your email — no credit card required
3. Complete onboarding and land on your Honeycomb home screen
4. **Copy your API key:** go to **Account → API Keys → Create API Key**
   - Name it `battleships-workshop`
   - Copy the key and save it somewhere (Notes app, etc.)

> You'll need this API key during the workshop. Don't lose it.

---

## 3. Download and install Kiro

Kiro is an AI IDE from AWS. You'll use it to generate your bot and add telemetry.

1. Go to **[Kiro.dev](https://kiro.dev)** and download the installer for your OS (Mac/Windows/Linux)
2. Install Kiro and open it
3. Sign in with your e-mail, Github or AWS account when prompted
4. Confirm you can see the chat/agent panel on the left side

> Kiro has a free tier with 50 credits available per month. Make sure it's working before the workshop.

---

## What you'll have ready on the day

- [ ] Honeycomb account created + API key saved
- [ ] AWS account active
- [ ] Kiro installed and signed in

**That's it.** Bring your laptop, a charger, and your competitive spirit.
